What is WikDict?

WikDict makes it easy to look up translations. It is a set of bilingual dictionaries with a simple web based search interface. One very important aspect of WikDict is that all data is freely available under a Creative Commons license, which is not the case for most of the competition.

Which data is available and where does if come from?

All data is extracted from Wiktionary by the DBnary project. So for data to be available, it has to be in one the the Wiktionaries support by DBnary. Since WikDict is still very young, not all DBnary data is uploaded to WikDict, yet, but that will change over time.

How do I search WikDict?
